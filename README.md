# GreenSkillsAI

Welcome to the **GreenSkillsAI** Training Repository!

## About the Session

This repository contains code, notebooks, and resources from the **GreenSkillsAI** training session, held at **SIT Nagpur** and organized by the **Edunet Foundation**.

- **Location:** SIT Nagpur  
- **Organizer:** Edunet Foundation  
- **Dates:** 2nd June 2025 – 17th June 2025

##  What We Learned

- Python programming fundamentals
- Data analysis with pandas and NumPy
- Data visualization using Matplotlib
- Real-world energy consumption datasets
- Building and interpreting charts (pie, bar, etc.)
- Hands-on coding exercises and collaborative learning

##  Repository Structure

- `*.ipynb` – Jupyter notebooks from the training
- `energy_consumption.csv` – Sample dataset used in exercises
- `README.md` – This file
